Given an array nums of integers, return the length of the longest arithmetic subsequence in nums.
Вам на вход поступает список целых чисел. Ваша задача найти самую длинную арифметическую прогрессию среди элементов этого списка и вывести её длину. 
Необязательно все элементы этого списка составляют арифметическую прогрессию, её может составлять любая из частей исходного списка, полученная путем удаления любого (в т.ч. нулевого) числа элементов из исходного списка без изменения порядка следования элементов.

*Пример 1:*
Ввод: 
3,6,9,12
Вывод:
4
Примечание: Весь список представляет из себя арифметическую прогрессию с шагом 3.

*Пример 2:*
Ввод:
9,4,7,2,10
Вывод:
3
Примечание: Самая длинная арифметическая прогрессия, полученная из исходного списка - [4,7,10].

*Пример 3:*
Ввод: 
20,1,15,3,10,5,8
Вывод:
4
Примечание: Самая длинная арифметическая прогрессия, полученная из исходного списка - [20,15,10,5].

*Пример 4:*
Ввод: 
1,1
Вывод:
2
Примечание: Весь список представляет из себя арифметическую прогрессию с шагом 0.


Ограничения:
Длина исходного списка: от 2х до 1000
Значения элементов в списке не превышают 1000
